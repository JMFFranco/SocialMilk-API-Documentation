Auth & Perfiles
Gestión de registro, login, sesión y administración de perfiles de usuario.
POST /auth/register Registrar perfil ▼
Crea un nuevo perfil. Retorna profileId y token JWT.
Request Body
{ "email": "usuario@ejemplo.com", "password": "MiPassword123", "phone": "3116176342", "fullName": "Juan Manuel Franco", "locationName": "Manizales" }
Parámetros
CampoTipoDescripción
emailstringCorreo electrónicorequerido
passwordstringContraseñarequerido
phonestringTeléfonorequerido
fullNamestringNombre completorequerido
locationNamestringCiudad/Ubicaciónrequerido
POST /auth/login Iniciar sesión ▼
Autentica al usuario. Retorna profileId y token JWT. Se puede usar email o teléfono.
Request Body
{ "email": "usuario@ejemplo.com", // o vacío "phone": "3116176342", // o vacío "password": "MiPassword123" }
POST /auth/logout 🔒 Cerrar sesión ▼
Invalida el token de sesión actual. Requiere Bearer token.
GET /auth/profiles/me 🔒 Mi perfil ▼
Retorna los datos del perfil del usuario autenticado.
PUT /auth/profiles/me 🔒 Actualizar perfil ▼
Request Body
{ "email": "", "phone": "3116176342", "fullName": "Juan Manuel Franco", "locationName": "Pereira" }
DELETE /auth/profiles/me Eliminar perfil ▼
Query Params
ParamTipoDescripción
profileIdstringID del perfil a eliminar
GET /auth/profiles/{profileId} 🔒 Perfil por ID ▼
Retorna el perfil público de cualquier usuario por su ID.

Animales
Gestión del hato ganadero. Registro, consulta, actualización y eliminación de animales.
GET /animals 🔒 Listar animales ▼
Query Params (opcionales)
ParamTipoDescripción
animalTypestringFiltrar por tipo (ej: "Vaca")
animalStatusstringFiltrar por estado (ej: "Produccion")
POST /animals 🔒 Crear animal ▼
Retorna el animalId del nuevo registro.
Request Body
{ "animalName": "Lola", "animalType": "Vaca", "animalBreed": "Holstein", "animalStatus": "Produccion", "birthDate": "2020-05-10" }
GET /animals/{animalId} 🔒 Animal por ID ▼
Retorna los datos completos de un animal específico.
PUT /animals/{animalId} 🔒 Actualizar animal ▼
Request Body
{ "animalName": "Lola Actualizada", "animalBreed": "Holstein Premium", "animalStatus": "Produccion" }
DELETE /animals/{animalId} 🔒 Eliminar animal ▼
Elimina un animal del hato por su ID.

Salud Animal
Historiales médicos y eventos sanitarios por animal. Todos los endpoints requieren autenticación.
Health Records — Historiales Clínicos
GET /animals/{animalId}/health-records 🔒 Listar registros ▼
Retorna todos los registros médicos del animal.
POST /animals/{animalId}/health-records 🔒 Crear registro ▼
Request Body
{ "recordType": "Vaccination", "description": "Aplicación vacuna aftosa" }
GET /animals/{animalId}/health-records/{recordId} 🔒 Registro por ID ▼
Retorna un registro médico específico.
PUT /animals/{animalId}/health-records/{recordId} 🔒 Actualizar registro ▼
Request Body
{ "recordType": "Treatment", "description": "Tratamiento veterinario actualizado" }
DELETE /animals/{animalId}/health-records/{recordId} 🔒 Eliminar registro ▼
Elimina un registro médico del historial.
Health Events — Eventos Sanitarios
GET /animals/{animalId}/health-events 🔒 Listar eventos ▼
Retorna todos los eventos sanitarios del animal.
POST /animals/{animalId}/health-events 🔒 Crear evento ▼
Request Body
{ "eventType": "Vaccination", "scheduledDate": "2026-06-01T10:00:00Z", "status": "PENDING" }
GET /animals/{animalId}/health-events/{eventId} 🔒 Evento por ID ▼
Retorna un evento sanitario específico.
PUT /animals/{animalId}/health-events/{eventId} 🔒 Actualizar evento ▼
Request Body
{ "eventType": "Checkup", "scheduledDate": "2026-06-10T08:00:00Z", "status": "COMPLETED" }
DELETE /animals/{animalId}/health-events/{eventId} 🔒 Eliminar evento ▼
Elimina un evento sanitario del animal.
GET /animals/health-events/upcoming 🔒 Eventos próximos ▼
Lista todos los eventos sanitarios próximos del hato, sin importar el animal. Útil para anticipar vacunas y tratamientos.

Interacciones en Posts
Comentarios y reacciones (likes) en publicaciones de la red social.
Comentarios
GET /posts/{postId}/comments 🔒 Listar comentarios ▼
Retorna todos los comentarios de una publicación. Soporta paginación con ?page=1&pageSize=10.
POST /posts/{postId}/comments 🔒 Crear comentario ▼
Request Body
{ "content": "Excelente producción de hoy." }
PUT /posts/{postId}/comments/{commentId} 🔒 Actualizar comentario ▼
Request Body
{ "content": "Comentario actualizado." }
DELETE /posts/{postId}/comments/{commentId} 🔒 Eliminar comentario ▼
Elimina un comentario de una publicación.
Likes
POST /posts/{postId}/likes 🔒 Dar like ▼
Registra un like del usuario autenticado en la publicación.
DELETE /posts/{postId}/likes 🔒 Quitar like ▼
Elimina el like del usuario autenticado en la publicación.
GET /posts/{postId}/likes 🔒 Ver likes ▼
Retorna el conteo y listado de likes de una publicación.

Insumos (Supplies)
Gestión de insumos, inventario y movimientos de entrada/salida/ajuste.
Suministros
POST /supplies 🔒 Crear insumo ▼
Request Body
{ "supplyName": "Concentrado Premium", "category": "Alimento", "unitMeasure": "Kg" }
GET /supplies 🔒 Listar insumos ▼
Retorna todos los insumos registrados.
GET /supplies/{supplyId} 🔒 Insumo por ID ▼
Retorna los detalles de un insumo específico.
PUT /supplies/{supplyId} 🔒 Actualizar insumo ▼
Request Body
{ "supplyName": "Concentrado Premium Plus", "category": "Alimento Bovino", "unitMeasure": "Kg" }
DELETE /supplies/{supplyId} 🔒 Eliminar insumo ▼

Inventario de Insumos
GET /supplies/inventory/supplies 🔒 Ver inventario ▼
Retorna el inventario completo de todos los insumos con sus cantidades actuales.
GET /supplies/inventory/supplies/{supplyId} 🔒 Inventario por insumo ▼

Movimientos de Insumos
POST /supplies/inventory/supplies/movements 🔒 Crear movimiento ▼
Soporta tres tipos de movimiento: IN (entrada), OUT (salida), ADJUSTMENT (ajuste).
Request Body
{ "supplyId": "{supplyId}", "movementType": "IN", // IN | OUT | ADJUSTMENT "quantityChanged": 100 }
GET /supplies/inventory/supplies/movements 🔒 Listar movimientos ▼
Query Params (opcionales)
ParamTipoDescripción

movementTypestringIN | OUT | ADJUSTMENT
GET /supplies/inventory/supplies/movements/{supplyMovementId} 🔒 Movimiento por ID ▼
Retorna los detalles de un movimiento de insumo específico.

Request Body

GET /ai/interactions 🔒 Historial de interacciones ▼
